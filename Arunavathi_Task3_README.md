# Random Password Generator

## Objective
A Python program that generates strong random passwords based on user-defined criteria.

## Features
- Generates passwords with user-defined length
- Supports uppercase letters, lowercase letters, numbers, and symbols
- Enforces minimum password length of 8 characters
- Validates user input
- Allows generating multiple passwords

## Technologies Used
- Python
- random module
- string module

## How to Run
1. Run the Python program.
2. Enter the required password length.
3. Select character types.
4. View the generated password.
