Basic Weather App

Objective

A Python program that fetches and displays real-time weather information for a user-specified city using the OpenWeather API.

Features

- Fetches real-time weather data
- Displays city name and country
- Shows temperature in Celsius
- Displays weather condition
- Shows humidity
- Displays wind speed
- Validates city name and displays an error message for invalid input

Technologies Used

- Python
- Requests module
- OpenWeather API

How to Run

1. Run the Python program.
2. Enter the city name.
3. View the weather details, including temperature, weather condition, humidity, and wind speed.
4. If an invalid city name is entered, the program displays an error message.
