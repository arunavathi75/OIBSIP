# BMI Calculator

## Objective
A Python program that calculates Body Mass Index (BMI) and classifies the result into health categories.

## Features
- Calculates BMI
- Displays BMI rounded to 2 decimal places
- Shows BMI category
- Validates invalid and negative inputs

## Technologies Used
- Python

## How to Run
1. Run the Python program.
2. Enter weight in kilograms.
3. Enter height in meters.
4. View your BMI and category.
